# Exercícios da Aula 1 de Desenvolvimento Web I - ADS
## lista_compras.md
- O exercício contém uma lista de compras que não está de forma odenada.
- Link: [https://github.com/anna-dev3148/web1_aula1/blob/main/lista_compras.md]
## lista_compras_numerada.md
- O exercício contém a mesma lista de compras do exercício anterior, porém ela está de forma ordenada.
- Link: [https://github.com/anna-dev3148/web1_aula1/blob/main/lista_compras_numerada.md]
## lista_compras_separada.md
- O exercício pede para que o aluno separe os itens por setor, no qual foi pedido a criação de 4 setores com 2 itens cada.
- Link: [https://github.com/anna-dev3148/web1_aula1/blob/main/lista_compras_separada.md]
## tabela_produtos.md
- O exercício pede para que o aluno pesquise 3 itens em dois sites diferentes e coloque em uma tabela os preços, com objetivo de aprender a fazer tabela e centralizações.
- Link: [https://github.com/anna-dev3148/web1_aula1/blob/main/tabela_produtos.md]
## artigo.md
- O exercício pede para que o aluno pesquise um artigo e marque duas palavras em negrito e duas em itálico.
- Link: [https://github.com/anna-dev3148/web1_aula1/blob/main/artigo.md]
